# drunk-drivers
Reinforcement learning to teach a humanoid robot to step out of a car in simulation

Install pybullet_robots

Clone repository
```
cd $path$
git clone git clone https://github.com/peconver/drunk-drivers.git
```

Run simulation
```
cd drunk-drivers
python3 play.py
```

Upload changes
```
cd drunk-drivers
git add $your_file$
git commit
git push
```

Download changes
```
git pull
```
